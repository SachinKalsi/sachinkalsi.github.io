# Images Folder

Place your blog post images in this folder.

## Usage in Markdown

Reference images in your blog posts like this:

```markdown
![Image description](/assets/images/your-image.jpg)
```

Or with HTML for more control:

```html
<img src="/assets/images/your-image.jpg" alt="Description" style="max-width: 100%;">
```

## Best Practices

- Use descriptive filenames (e.g., `machine-learning-diagram.png` instead of `img1.png`)
- Optimize images for web (compress large images)
- Use appropriate formats:
  - `.jpg` or `.jpeg` for photos
  - `.png` for graphics with transparency
  - `.gif` for animations
  - `.webp` for modern browsers (better compression)
